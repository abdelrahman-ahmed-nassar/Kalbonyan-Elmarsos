
  2- boolean data type

      bool isLightOn = false;

  3- character data type  
    char mean one letter or symbol   
    take 8 bit or 16 in other language 

    Ex:  (C++)
      char firstInitial = 'K';
      Take (8 bits)

    Ex:   (Kotlin)
      val firstInitial: char = 'K';
            Take (16 bits)
